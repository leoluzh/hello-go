# Code of Conduct

All contributors are expected to follow a standard code of conduct to foster an open and welcoming environment.

This project adopts the Contributor Covenant v2.1 as its Code of Conduct. For the full text and details, see: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

Reporting
- If you experience or witness unacceptable behavior, please contact the maintainers via the repository's GitHub Security Advisories or open a private Security Advisory (see SECURITY.md).

Enforcement
- Reports will be handled promptly and may result in temporary or permanent removal from the project depending on severity.
